# Franco Moreira

**Agro · Código · Sonido**

Intersección entre tecnología, agricultura y construcción sustentable.

La tecnología y la IA no son solo para startups millonarias ni campos inteligentes con sensores de 500 dólares. También pueden servir para la huerta familiar, el productor chico, el que labura con lo que tiene. Una aplicación web que ayuda a rotar cultivos, un asistente que corre en una laptop vieja, un sensor hecho con un microcontrolador de 5 dólares. La pregunta no es qué tecnología existe, sino cómo adaptarla a los recursos que hay.

---

### Áreas de interés

**🌱 Permacultura y bioconstrucción** — Sistemas que cierran ciclos. Masa térmica, captura de agua, invernaderos pasivos, Earthship híbrido. Lo que el territorio da y lo que necesita.

**🎸 Música y producción** — Grabar, producir, tocar. El escenario puede ser un estudio o un fogón.

**🛠️ Dev** — Herramientas que resuelven problemas reales. HTML, CSS, JavaScript, React, Astro, algo de backend. No por hacer tecnología, porque sirve para algo.

**🤖 IA aplicada** — Asistentes de terminal, automatización, agentes locales. Que la máquina se adapte a uno, no al revés.
